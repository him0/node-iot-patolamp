# IoT PATOLAMP (Node IoT PATOLAMP)

Manage Patolamp state with AWS IoT Core.

## Setup

```
$ npm install -g pm2
$ npm install
```

## Run Server

```
$ pm2 start ecosystem.config.js
```

## Run Client

```
$ node client.js
```
